The website features photos from Pixaby.com. The key to animal photography is patience. It can take a long time to get the right lighting and the best view of the animal. When photograghping animals, you do not have a lot of control...you need to wait for them.

Web Design Exercise

Animal Photography is a website exercise on image galleries available on WebDevStudents.com. The images are royalty-free photos from pixabay.com. JavaScript was used to make the photo gallery, and Bootstrap columns for responsive layout.
